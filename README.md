# 🤖 Smart Assistant App

An Android virtual assistant that enables users to perform everyday tasks using voice commands.
The application leverages Google's ML Kit for speech recognition and follows the MVVM architecture to provide a scalable, maintainable, and responsive user experience.

---

## ✨ Features

- 🎤 Voice command recognition
- 📞 Make phone calls using voice
- 💬 Send SMS using voice
- 🔍 Voice search
- 📍 Current location detection
- 🗺️ Google Maps integration
- 🔐 Firebase Authentication
- 💾 Local data storage with Room
- ⚡ Asynchronous programming using Coroutines

---

## 🛠 Tech Stack

- Kotlin
- Java
- MVVM Architecture
- Android Jetpack
- ML Kit
- Room Database
- Coroutines
- Firebase Authentication
- Google Maps API
- Fused Location Provider

---

## 📱 Future Improvements

- AI chatbot integration

- Gemini/OpenAI integration
- Weather updates
- Calendar reminders
- Smart home device control

---

## 🚀 Getting Started

1. Clone the repository.
2. Open the project in Android Studio.
3. Add your Firebase configuration.
4. Add a Google Maps API key.
5. Sync Gradle and run the application.
